# SKILL Event Website (Static)

Folder ini berisi website statis untuk event Seminar Keilmiahan dan Hasil Penelitian (SKILL).

## Struktur
- index.html: halaman utama
- assets/css/styles.css: styling
- assets/js/main.js: interaksi sederhana (nav, reveal, FAQ)
- assets/images/: gambar placeholder

## Cara mengganti konten
- Ubah teks di index.html (judul, tanggal, lokasi, deskripsi).
- Ganti gambar di assets/images/ dengan foto asli, lalu sesuaikan src di index.html.
- Atur warna, layout, dan tipografi di assets/css/styles.css.

## Catatan
Form registrasi dan kontak masih placeholder. Hubungkan ke backend atau layanan form yang Anda gunakan.
